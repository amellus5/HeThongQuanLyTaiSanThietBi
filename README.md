# HỆ THỐNG QUẢN LÝ TÀI SẢN VÀ THIẾT BỊ

## Giới thiệu
Đây là dự án đồ án môn học xây dựng hệ thống quản lý tài sản và thiết bị.
Hệ thống giúp quản lý thông tin thiết bị, nhân viên và quá trình mượn – trả thiết bị trong tổ chức.

## Chức năng chính
* Quản lý nhân viên
* Quản lý loại thiết bị
* Quản lý thiết bị
* Quản lý phiếu mượn – trả thiết bị
* Báo cáo và thống kê

## Công nghệ sử dụng
* HTML, CSS, JavaScript (xây dựng giao diện website)
* Antigravity (hỗ trợ xử lý logic và quản lý dữ liệu)
* Visual Studio Code (môi trường phát triển)
* Netlify (triển khai website online)
